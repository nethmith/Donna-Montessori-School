# Donna Montessori

This workspace is a React app built with Vite.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Run lint checks:

```bash
npm run lint
```

## Project Notes

- React routes and shared layout live under [src](src).
- Static assets live under [public](public).
- The app uses Vite, React Router, React Helmet Async, Axios, and Tailwind CSS utilities.
